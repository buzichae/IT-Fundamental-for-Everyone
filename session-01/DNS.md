## DNS

마지막으로 **DNS**입니다.

IP 주소에는 문제가 있습니다.

1️⃣ 기억하기 어렵다

2️⃣ 변경될 수 있다

그래서 등장한 것이 **DNS (Domain Name System)** 입니다.

DNS는 쉽게 말하면

> **인터넷 전화번호부**
> 

입니다.

예를 들어

```
google.com
```

을 입력하면

DNS 서버가

```
google.com → IP 주소
```

로 변환해 줍니다.

그래서 우리는

복잡한 IP 주소 대신

**도메인 이름을 사용해서 서비스에 접근할 수 있습니다.**

---

## 정리

지금까지 내용을 정리해 보면 다음과 같습니다.

- **IP** → 인터넷에서 데이터를 전달하는 기본 규칙
- **TCP** → 신뢰성 있는 통신 제공
- **UDP** → 빠른 통신을 위한 단순한 프로토콜
- **Port** → 하나의 컴퓨터에서 애플리케이션 구분
- **DNS** → 도메인을 IP 주소로 변환
